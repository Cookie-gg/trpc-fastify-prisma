# trpc-fastify-prisma

Basic CRUD application with trpc, fastify, and prisma.

## Getting Started

### Install dependencies

```bash
yarn
```

### Database setup

```bash
yarn -W db:start
```

### Generate prisma client & migrations

```bash
yarn -W prisma
```

### Run tRPC server

```bash
yarn -W run dev
```

### Run client app

```bash
yarn workspace playground run dev
```
